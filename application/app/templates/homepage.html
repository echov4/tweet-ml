Introduction

This project aims to classify tweets as offensive or not offensive using various machine learning models. The models used in this project include Logistic Regression, Decision Trees, Random Forest, and Gradient Boosting Classifier.


To understand more about the project and how it was planned and broken down, visit the Project Explained tab on the navigation bar The twitter classification program was created by myself to allow for other users to test if a tweet is considered as offensive or not.

If you wish to test the models against a custom tweet, click on the Classify Tweet tab on the navigation bar.
